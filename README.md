\# Ordered Logistic Regression Modeling of Shaking Table Separation



This project investigates statistical modeling of particle separation in shaking tables using MLA particle data.



\## Objective

To identify particle characteristics influencing classification into density-based streams using regularized ordinal logistic regression.



\## Methodology

\- Data preprocessing and feature engineering

\- Elastic-net regularized ordinal regression using the `ordinalNet` package

\- Hyperparameter tuning via cross-validation

\- Bootstrap analysis for coefficient stability



\## Key Findings

Particle density was identified as the dominant predictor of separation behavior, with geometric particle features acting as secondary modifiers.



\## Technologies

\- R

\- ordinalNet

\- parallel computing
